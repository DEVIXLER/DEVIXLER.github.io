Command List

Last updated: November 27, 2024

Information

  • &about
  • &help: this exact thing
  • &disc: my discord server
  • &ping: Pong!
  • &test: tests if the system is up and running
  • &time: tells you the time

Interactive

  • &abc: converts from any arbatrary base to base10
  • &script: runs any JavaScript you give it!
  • &reverse: reverses a string
  • &echo: says the exact string you give it
  • &hash: hashes a string (e.g., &hash QISMKXS)
  • &color: same as hash, just more fancy for color use
  • &pi: type any index of pi you want (up to 100 trillion digits!) (NOT WORKING)
  • &tms: gets the Thue-Morse Sequence at any index
  • &fraction: converts a float to a fraction (~85% accuracy)
  • &cdsim: a cooldown simulator (9-minute countdown)
  • &otp: one-time pad encryption algorithm (still broken)
  • &dn: toggles "deeznuts trolling" mode
  • &phoneword: converts words or sentences to keypad numbers

Games

  • &number: guess the number (currently broken, avoid using)
  • &clock: guess the time on a randomly generated clock

Chat

  • &prevmsg: shows the last 10 messages sent in chat (shortcut: &p)
  • &boner: allows you to see previous messages at a certen index from the bot's logs, you can see the index of current messages with &p
